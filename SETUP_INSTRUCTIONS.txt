**HOW TO RUN THIS PROJECT IN VS CODE**
===============================================================================

**PREREQUISITES:**
1. **Node.js** (Download from: https://nodejs.org/)
2. **VS Code** (Download from: https://code.visualstudio.com/)
3. **MongoDB** (Download from: https://www.mongodb.com/try/download/community)

===============================================================================

**STEP 2: INSTALL REQUIRED PACKAGES**
===============================================================================

===============================================================================

**In VS Code terminal, run:**

**npm run dev**

**http://localhost:5173**

===============================================================================
**STEP 5: MONGODB SETUP (LOCAL)**
===============================================================================

**Option 1: Install MongoDB Locally**

1. **Download MongoDB Community Server**
2. **Install MongoDB**
3. **Start MongoDB service:**
   - **Windows:** **net start MongoDB**
   - **Mac/Linux:** **sudo systemctl start mongod**

4. **Default MongoDB URL:**
   **mongodb://localhost:27017/bootcamp**

**Option 2: Use MongoDB Atlas (Cloud)**

1. **Create account at:** **https://www.mongodb.com/atlas**
2. **Create cluster**
3. **Get connection string**
4. **Paste in the MongoDB section of the app**

**STEP 8: ENVIRONMENT VARIABLES SETUP**
===============================================================================

**Create a .env file in your project root:**

```
**NODE_ENV=development**
**PORT=5000**
**MONGODB_URI=mongodb://localhost:27017/bootcamp**
**JWT_SECRET=your_jwt_secret_key**
**FRONTEND_URL=http://localhost:5173**
```

**Install dotenv to use environment variables:**
**npm install dotenv**

```json
    "start": "node server/server.js"
  }
}
```

===============================================================================
**STEP 10: TROUBLESHOOTING COMMON ISSUES**
===============================================================================

**Issue: Port already in use**
- **Solution:** **npx kill-port 5173** or **npx kill-port 5000**

**Issue: Module not found**
- **Solution:** **npm install** or **npm install [package-name]**

**Issue: MongoDB connection error**
- **Check if MongoDB service is running**
- **Verify connection string**
- **Check firewall settings**

**Issue: React app not updating**
- **Clear browser cache: Ctrl + Shift + R**
- **Restart dev server: Ctrl + C, then npm run dev**
